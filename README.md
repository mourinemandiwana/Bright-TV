# Bright-TV

- BrightTV-User-Consumption-Analysis

- This project focuses on analysing the “BrightTV Viewership” dataset, which contains user profile information and session-based viewing activity. The analysis was conducted to support the CEO’s objective of growing the platform’s subscription base and improving overall user engagement.

- As a Data Analyst, the objective was to extract meaningful insights on user behaviour, viewing patterns, and content consumption, and provide actionable recommendations to support the Customer Value Management (CVM) team.

- Objective

- The analysis aimed to answer the following key business questions:

     - What are the key user and usage trends on BrightTV
     - What factors influence user consumption behaviour
     - Which days and times have low consumption, and how to improve them
     - What strategies can be implemented to grow the user base
 
       
- Tools and Technologies
     - Databricks: Data cleaning, transformation, and SQL analysis
     - SQL: Data aggregation, joins, and feature engineering
     - Excel / Power BI: Data visualisation and dashboarding
     - Miro: Data planning and structuring the analytical approach
     - Gantt Chart: Project planning and timeline tracking
 
       
- Data Preparation
     - Joined user profiles and viewership datasets using UserID
     - Converted timestamps from UTC to South African Time (UTC +2)
     - Created calculated fields:
     - Age classification (Kids, Teenagers, Young Adults, Adults)
     - Engagement levels based on session duration
     - Time buckets (Midnight, Morning, Afternoon, Evening)
     - Extracted date features such as:
     - Month
     - Day of week
     - Day of month
 
       
- Key Insights
     - Peak consumption occurs during evening hours, indicating prime-time viewing behaviour
     - Low engagement periods are mostly during early morning (midnight–5am)
     - Adults and young adults contribute the most to total viewership
     - Certain channels/content types drive higher engagement, suggesting strong content preferences
     - Session duration is a strong indicator of user engagement and retention
     - Viewing patterns vary across days of the week, with noticeable dips on specific days
     - Factors Influencing Consumption
     - Time of day (prime-time vs off-peak hours)
     - Age group and demographic profile
     - Content type / channel popularity
     - Session duration (engagement level)
     - Day of the week (weekday vs weekend behaviour)
 
       
- Recommendations
     - Introduce targeted content scheduling during low-consumption periods (e.g. morning shows, short-form content)
     - Promote high-performing channels/content to increase engagement
     - Personalise recommendations based on user demographics and viewing history
     - Launch weekday engagement campaigns to boost low-performing days
     - Optimise content for mobile-friendly, short-duration viewing during off-peak hours
       
- Growth Initiatives
     - Offer free trials or discounted subscription bundles to attract new users
     - Implement referral programmes to encourage existing users to bring in new subscribers
     - Partner with local content creators to diversify and localise content
     - Introduce loyalty or rewards programmes based on viewing activity
     - Use data-driven marketing campaigns targeting high-potential user segments
 
       
- Business Impact

- This analysis provides actionable insights that can help BrightTV:

     - Increase user engagement and viewing time
     - Improve content strategy and scheduling
     - Grow the subscription base
     - Enhance customer retention
     - Enable data-driven decision-making for CVM and leadership
